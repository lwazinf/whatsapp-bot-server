## Implementation Notes

### Database Storage
```typescript
await prisma.platformTerms.create({
  data: {
    version: "1.0",
    customerTerms: CUSTOMER_TERMS_TEXT,
    merchantTerms: MERCHANT_TERMS_TEXT,
    privacyPolicy: PRIVACY_POLICY_TEXT,
    liabilityWaiver: LIABILITY_WAIVER_TEXT,
    platformLiabilityClause: PLATFORM_LIABILITY_CLAUSE,
    merchantLiabilityClause: MERCHANT_LIABILITY_CLAUSE,
    isActive: true,
    effectiveDate: new Date(),
    createdBy: ADMIN_USER_ID
  }
});
```

### WhatsApp Display
```typescript
// Break into chunks (WhatsApp limit: 4096 chars)
const chunks = splitIntoChunks(terms, 3800);
for (const chunk of chunks) {
  await whatsapp.sendText(phone, chunk);
  await sleep(1000); // Prevent rate limiting
}
```

### Tracking Acceptance
```typescript
// Log acceptance with timestamp and IP
await prisma.termsAcceptance.create({
  data: {
    userId: user.id,
    businessId: business?.id,
    termsVersion: "1.0",
    acceptedAt: new Date(),
    ipAddress: getIPAddress(request),
    userAgent: request.headers['user-agent']
  }
});
```
